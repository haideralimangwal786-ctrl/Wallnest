# 🎧 WallNest – Wallpaper App (Flutter + Pexels API)

WallNest ek modern Flutter-based wallpaper application hai jo Pexels API se high-quality wallpapers fetch karta hai. Is app mein users wallpapers search kar sakte hain, categories browse kar sakte hain aur apne favorite wallpapers save kar sakte hain.

---

# 🚀 Features

- 🔍 Wallpaper search system (Pexels API)
- 📂 Category-based browsing
- 🖼️ Full-screen wallpaper preview
- ⭐ Add / remove favorites
- 💾 Offline storage (SharedPreferences)
- ⚡ Fast image loading
- 📱 Responsive UI (Mobile friendly)

---

# 🛠️ Tech Stack

- Flutter (UI Framework)
- Dart (Programming Language)
- Pexels API (Images Source)
- HTTP Package (API Calls)
- SharedPreferences (Local Storage)

---





# 💻 How to Run This Project

## 1. Clone Repository
git clone https://github.com/your-username/wallnest.git

## 2. Open Project
Open in VS Code or Android Studio

## 3. Install Dependencies
flutter pub get

## 4. Run Project
flutter run

---

# 📱 Build APK (Optional)

flutter build apk --release

APK file location:
build/app/outputs/flutter-apk/app-release.apk
